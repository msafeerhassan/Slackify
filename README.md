# Slackify

## A personal assistant Slack Bot

### Coded in JavaScript

#### Features:
1. 20+ Commands Available
2. AI Integration
3. Uses Completely free API's
4. Gives a bit sarcastic response if any command fails
5. Commands available for every type of mood :)

#### How to test it?
1. Go over to any channel in Hack Club Slack Workspace - preferably a private or personal channel - we have a dedicated channel named `#slackify-bot` for testing.
2. Simply test the commands :)

#### How to use it locally?
1. Clone this repository.
2. Install the required packages including node.
3. Add your own API Keys and credentials in .env file.
4. Run `node index.js`

#### Available Commands

| :--- | :--- | :--- |

| `/slackify-dogfact` | `/slackify-dogfact` | Fetches a random dog fact |
| `/slackify-joke` | `/slackify-joke` | Fetches a random joke |
| `/slackify-tip` | `/slackify-tip` | Fetches a random tip from HCAI |
| `/slackify-quote` | `/slackify-quote` | Fetches a random Quote |
| `/slackify-meme` | `/slackify-meme` | Fetches a random meme (less funny than your life) |
| `/slackify-advice` | `/slackify-advice` | Fetches a random piece of Advice |
| `/slackify-dogpic` | `/slackify-dogpic` | Fetches a random dog picture |
| `/slackify-foxpic` | `/slackify-foxpic` | fetches a random fox pic |
| `/slackify-bored` | `/slackify-bored` | Fetches a random activity suggestion when you are bored |
| `/slackify-numtrivia` | `/slackify-numtrivia` | Fetches a Number Trivia |
| `/slackify-affirm` | `/slackify-affirm` | Fetches a random affirmation |
| `/slackify-toss` | `/slackify-toss` | Fetches either yes or no + an image |
| `/slackify-weather` | `/slackify-weather Cityname` | Fetches live weather of specific city |
| `/slackify-github` | `/slackify-github username` | Fetches github Stats of a specific Github User |
| `/slackify-excuse` | `/slackify-excuse` *(category is optional)* | Fetches an excuse based on your given category |
| `/slackify-define` | `/slackify-define indeed` | Fetches definition of specific given word |
